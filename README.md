# msnap
Refusing the Noise: Adaptive Null-Space Projection for Mitigating Susceptibility to Misinformation in Multi-Document RAG


## Install Guide

Anaconda 가상 환경 생성
```bash
conda create -n msnap python=3.11.7 -y
```

라이브러리 설치
```bash
pip install nvidia-ml-py PyYAML openai
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121
```
